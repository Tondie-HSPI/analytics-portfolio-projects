# Analytics Portfolio Projects

This repository contains curated analytics projects that demonstrate R, SQL, statistical modeling, database design, and business recommendation skills.

## Projects

### Mobile Ad Click Prediction in R

Uses CART and XGBoost to predict mobile ad click behavior, evaluate model performance, and recommend a higher-ROI targeting strategy.

Folder: `portfolio-projects/mobile-ad-click-prediction-r`

### Promotion ROI Analysis in R

Evaluates temporary price reductions, display support, feature support, lift factors, and forward buying in a trade promotion setting.

Folder: `portfolio-projects/promotion-roi-analysis-r`

### Product Pricing and Elasticity Analysis in R

Estimates own-price and cross-price elasticities to analyze cannibalization, competitive threat, and product-line pricing decisions.

Folder: `portfolio-projects/product-pricing-elasticity-r`

### Consumer Segmentation with PCA in R

Uses principal component analysis and consumer mapping to identify interpretable market segments from survey response data.

Folder: `portfolio-projects/consumer-segmentation-pca-r`

### SQL Database Design Portfolio

Shows relational database modeling and SQL query examples for business-style expense analysis.

Folder: `portfolio-projects/sql-database-design-portfolio`

## Notes

Original source datasets are omitted where they are private, academic, or not appropriate for public redistribution. Each project includes a data note and a README explaining the analysis workflow, methods, and business value.
