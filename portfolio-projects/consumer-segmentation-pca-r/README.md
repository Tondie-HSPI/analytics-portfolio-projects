# Consumer Segmentation with PCA in R

This project uses principal component analysis to identify consumer attitude dimensions and translate survey responses into actionable market segments.

## Business Question

How can survey responses be reduced into meaningful consumer segments for marketing strategy?

## Tools

- R
- R Markdown
- PCA
- Correlation analysis
- Consumer mapping
- Segment interpretation

## Methods

- Imported and scaled survey response data.
- Created a correlation matrix and correlation plot.
- Performed PCA to reduce five attitude variables into core dimensions.
- Interpreted factor loadings and named the principal components.
- Built a two-dimensional consumer map and described market segments.

## Key Findings

- Two principal components captured most of the variation in customer attitudes.
- The analysis identified service-expectation and value-orientation dimensions.
- Consumers could be grouped into interpretable marketing segments.
- Segment-level interpretation supports positioning and targeting decisions.
