# SQL Database Design Portfolio

This project highlights relational database design and SQL analysis work in MySQL Workbench. It is based on local schemas including expense, university, Sakila, and other practice databases.

## Business Question

How can relational data be structured and queried to support operational and business analysis?

## Tools

- MySQL
- MySQL Workbench
- SQL
- Entity relationship modeling
- Joins, aggregation, filtering, and schema inspection

## Portfolio Highlights

- Designed and explored normalized database schemas.
- Worked with primary keys, foreign keys, table relationships, and schema creation.
- Queried business-style expense data across employees, departments, accounts, trips, reimbursements, and reason codes.
- Used MySQL Workbench to inspect and model relational database structure.

## Files

- `portfolio_queries.sql` - sample SQL queries written for portfolio demonstration.
- `assets/mysql_workbench_model.mwb` - MySQL Workbench model file.

## Portfolio Note

Instructor PDFs, packaged course files, and seed datasets are intentionally omitted. This public version focuses on database design concepts and original query examples that can be adapted to a local MySQL environment.
