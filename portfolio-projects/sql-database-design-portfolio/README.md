# SQL Database Design and Data Warehousing Portfolio

This project highlights MySQL database design, query development, and data warehousing work from CIS467: Data Management, Data Warehousing, and Data Visualization.

## Business Question

How can operational databases be structured, queried, and transformed into analytics-ready tables for business decision-making?

## Tools

- MySQL
- MySQL Workbench
- SQL
- EER/database modeling
- Data warehousing
- Aggregation, joins, subqueries, views, and reusable functions

## Portfolio Highlights

- Wrote SQL queries for operational safety analytics, including incident-rate reporting by location and division.
- Designed an expense data warehouse by joining employee, trip, department, reason-code, and expense tables.
- Built warehouse-level queries for department spend, trip purpose, above-average travel costs, and employee spending.
- Designed a film-rental performance warehouse using the Sakila database.
- Created film-performance metrics including revenue, rental count, customer reach, inventory depth, revenue per rental, and rentals per copy.
- Added reusable SQL classification logic to segment films into performance tiers.

## Files

- `portfolio_queries.sql` - curated SQL portfolio file covering safety analysis, expense warehousing, and film-rental performance analysis.
- `assets/mysql_workbench_model.mwb` - MySQL Workbench model file supporting the database design work.

The public version is curated for readability and review.
